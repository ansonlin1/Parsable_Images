# Parsable_Images

Collecting images from the Parsable database using the Parsable API.

To Run (On Windows):
Download Python3
Download Git Bash
Install/Update PIP
Install Requests
Create Directory on Desktop called "PythonImages"
Run: python Images_Grab_Lip_Print.py

To Run (On Mac):
Download Python3
Install/Update PIP
Install Requests
Create Directory on Desktop called "PythonImages"
Run: python Images_Grab_Lip_Print.py
